## Notas
### ¿Qué es ROT13?

- **ROT13 (Rotate by 13)** es un tipo de cifrado por sustitución.
- Cada letra del alfabeto se reemplaza por la que está **13 posiciones adelante**.
- Es un caso especial del **cifrado César**.

Ejemplo:

- A → N
- B → O
- C → P
- …
- N → A

Es simétrico: aplicar ROT13 dos veces regresa al texto original.
