## Solucion
El objetivo del ejercicio es **corregir errores de sintaxis en un programa escrito en Rust** para que compile correctamente y muestre la flag.

- Se proporciona un proyecto comprimido (`.tar.gz`)
- Contiene un proyecto con **Cargo**
- El programa tiene errores intencionales
- Deben corregirse usando los mensajes del compilador
#### Objetivo

- Identificar errores de compilación
- Corregirlos
- Ejecutar el programa correctamente
- Obtener la flag
#### Herramientas utilizadas

- `cargo run` → compilar y ejecutar
- `rustc` → compilador de Rust
- Editor de texto (`nano`, `vim`, VS Code)
