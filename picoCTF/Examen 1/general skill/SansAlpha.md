## SansAlpha

## Descripcion
The Multiverse is within your grasp! Unfortunately, the server that contains the secrets of the multiverse is in a universe where keyboards only have numbers and (most) symbols.`ssh -p 50370 ctf-player@mimas.picoctf.net`Use password: `1db87a14`
## Solucion
- Conectarse por SSH
- Generar errores → obtener letras
- Guardar errores en variables
- Extraer caracteres
- Usar wildcards para encontrar archivos
- Construir comandos (`cat`, `echo`)
- Leer la flag

**picoCTF{7h15_mu171v3r53_15_m4dn355_4945630a}**

## Notas
- Manipulación avanzada de Bash
- Uso de variables y substrings
- Redirección de errores (`2>&1`)
- Técnicas de evasión de restricciones
- Uso de globbing (`*`, `?`)
